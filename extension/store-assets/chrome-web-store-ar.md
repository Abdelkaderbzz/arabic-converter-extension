# Arabizzi — نصوص Chrome Web Store (العربية)

## الوصف المختصر

حوّل Arabizi والعامية من الحروف اللاتينية إلى العربية أو الفصحى — من شريط Chrome مباشرة.

## الوصف التفصيلي

**Arabizzi** إضافة Chrome مجانية تحوّل النص بالعربية العامية (Arabizi) إلى:

- **عامية** — نفس المعنى بخط عربي
- **فصحى** — العربية الفصحى الحديثة

### المميزات

- لا تحتاج لوحة مفاتيح عربية (`3`→ع، `7`→ح، `9`→ق)
- واجهة عربية وإنجليزية مع RTL
- نسخ، سجل التحويلات، والمحفوظات
- مفتاح Gemini API مجاني (يُخزَّن محلياً)

**الموقع:** [arabizzi.com](https://arabizzi.com/)
